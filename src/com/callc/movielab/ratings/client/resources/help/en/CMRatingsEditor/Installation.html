Installation

Requirements

The Editor is Java-based. The user's runtime environment must, therefore, include a Java JRE in the path. The minimum recommended JRE is 1.6.

All required supporting jars are provided with the distribution. Generation of HTML from the XML specifications does, however, require that the user obtain a valid license for Saxon. This is available at http://www.saxonica.com/.  Either the Professional (PE) or Enterprise (EE) license may be used. The minimum required version is 9.5.

The Editor may be used in either a Windows or Linux environment and both BAT and BASH scripts have been provided for starting and running the editor.

Installing

Unpack the distribution into any directory. To run the editor, simply cd to the install directory and run either the BAT or BASH script.
