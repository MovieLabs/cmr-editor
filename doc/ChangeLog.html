Change Log and Revision History

  • 2013-Nov-14:

    • Changes to XSD schema:
      • Specific Ratings may now be associated with specific Reason codes.
      • Entire Rating System may be tagged as DEPRECATED
      • Multiple Regions of usage may be associated with a single Rating System

    • Changes to Specifications:
      • The Icelandic SMAIS rating system has been tagged as DEPRECATED. Iceland has been added to the Dutch NICAM (a.k.a. Kijwijzer) rating system as an additional region of usage.

    • Other Changes:

  • 2013-Nov-07:

    • Changes to XSD schema:
      • Multiple symbols or icons may be specified for a given Rating
      • Symbols or icons may now be specified for a Reason

    • Changes to Specifications:
      • Reasons and Content Description classifiers have been added for the following Rating Systems:
        • Netherlands - Kijwijzer
        • United States - FAB
        • United States - TVPG
      • Additional symbols have been added to US-MPAA ratings.

    • Other Changes:
      • XSD is now available on-line
      • Layout and contents of the HTML page for each Rating System has been changed significantly.
      • Reasons and Content Description classifiers are not listed on the Summary page.

  • 2013-Oct-30:

    • Changes to XSD schema:
      • Addition of HPCApplicable element to Description elements. This indicates if the Rating is applicable to usage in a home Parental Control system. This is intended as a hint for systems such as UltraViolet and is not intended as a requirement regarding the usage of the Rating in a HPC implementation.
      • Addition of an optional Explanation element to Description elements

    • Changes to Specifications:
      • All existing Rating Systems have been updated to conform with XSD changes described above.
      • HTML for each Rating has been modified to show a short label, long label, definition and (optional) explanation. The ordinal value is not shown in the HTML but is present in the XML.
      • Addition of deprecated ratings M and GP to US MPAA.
      • Correction to ordinal values for US FAB ratings.

  • 2013-Oct-23:
    • AU NCS: Removal of spaces from value field of MA 15+, R 18 +, and X 18+
    • Addition of ordinal attribute to XML specification for all existing ratings systems
    • New ratings systems added:
      • US TVPG
      • US MPAAT
